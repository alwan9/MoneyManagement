Prompt 1 — Setup Project

Buatkan project Google Apps Script menggunakan HTML, CSS, JavaScript dan Tailwind CSS. Terapkan arsitektur modular sesuai struktur folder. Jadikan aplikasi sebagai Progressive Web App (PWA) lengkap dengan manifest.json dan service-worker.js agar dapat diinstal di Android, iPhone, dan desktop. Gunakan desain modern, clean, premium, responsive, dan mobile-first.

Prompt 2 — Database

Buat struktur Google Spreadsheet sebagai database dengan sheet Users, Trading, Finance, Permission, dan Setting. Semua data Trading dan Finance wajib memiliki kolom UserID sebagai relasi. Buat fungsi Apps Script untuk membaca, menambah, mengubah, dan menghapus data.

Prompt 3 — Login

Buat sistem login multi-user menggunakan Google Apps Script. Password harus di-hash, menggunakan session login, dan setiap pengguna memiliki UserID unik. Setelah login berhasil, tampilkan hanya data yang sesuai dengan UserID pengguna tersebut.

Prompt 4 — Role & Permission

Implementasikan dua role, yaitu User dan Super Admin. User hanya dapat mengakses data miliknya sendiri. Super Admin dapat mengelola seluruh user, seluruh data, serta mengatur permission melalui halaman admin. Permission disimpan di Spreadsheet dan divalidasi di backend sebelum setiap proses CRUD dijalankan.

Prompt 5 — Dashboard

Buat halaman Dashboard dengan ringkasan statistik trading dan keuangan, kartu informasi (Total Profit, Total Loss, Win Rate, Saldo, Pengeluaran), grafik perkembangan, dan aktivitas terbaru. Desain menggunakan Tailwind CSS dengan tampilan profesional.

Prompt 6 — Progress Trading

Buat modul Progress Trading lengkap dengan fitur CRUD, tabel data, pencarian, filter, pagination, modal input/edit, upload screenshot, statistik sederhana, dan integrasi TradingView Widget gratis sebagai tampilan chart.

Prompt 7 — Kelola Keuangan

Buat modul Kelola Keuangan dengan fitur CRUD, kategori transaksi, pemasukan/pengeluaran, pencarian, filter, pagination, serta ringkasan saldo dan pengeluaran. Semua data harus terhubung ke Spreadsheet dan difilter berdasarkan UserID.

Prompt 8 — Penyempurnaan

Rapikan seluruh project dengan clean code, modular, reusable component, validasi frontend dan backend, loading state, toast notification, responsive layout menggunakan Tailwind CSS, serta optimasi performa Google Apps Script. Pastikan seluruh fitur berjalan tanpa error dan siap digunakan sebagai aplikasi produksi.
